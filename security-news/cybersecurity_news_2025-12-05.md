# 2025年12月5日 サイバーセキュリティニュースまとめ
## 主要ポイント
* **PRC国家アクターによる仮想化基盤（VMware vCenter/vSphere）への戦略的標的化**と、BRICKSTORMマルウェアを用いた重要インフラへの永続的アクセス確立が、本日最大の脅威としてCISAより警告された,,。
* CVSSスコア10.0の**React Server Components RCE脆弱性（CVE-2025-55182）**に対する有効なPoCが公開され、中国系アクターが迅速かつ広範な悪用を開始している,,,。
* LLM駆動型エージェントの「過剰なエージェンシー」を悪用し、ユーザーの確認なしにGoogle Driveのデータを削除する**Zero-Click Agentic Browser Attack**が発見され、AI駆動型システムの新たなセキュリティリスクが浮上した,,。

## 国家主導型脅威と仮想基盤標的化
2025年12月5日にCISA（サイバーセキュリティ・インフラストラクチャセキュリティ庁）、NSA、カナダサイバーセキュリティセンターが合同アドバイザリを公開し、中華人民共和国（PRC）国家支援型アクターが使用する高度なバックドアマルウェア「BRICKSTORM」の脅威を警告した,,。BRICKSTORMは、現代エンタープライズITインフラストラクチャの仮想化レイヤーである**VMware vSphereおよびvCenterサーバー**を主要な標的としており,、攻撃者はネットワーク防御の中心を掌握することが可能となる。このマルウェアは通信を隠蔽し、ネットワーク内で横方向に移動し、サービス中断時でも自動的に再インストールする持続性を有している,。CrowdStrikeの報告では、BRICKSTORMを展開するWarp Pandaアクターが、政府、ITセクターに加え、**アジア太平洋地域**の法律、技術、**製造業**を含む複数の業界を標的としていることが確認されている,,,。

## ゼロデイ脆弱性の緊急悪用とインシデント連鎖
Webアプリケーション開発の主要コンポーネントであるReact Server Componentsに存在する深刻なRCE脆弱性（CVE-2025-55182、CVSSスコア10.0）について、有効な概念実証（PoC）コードが公開され、JPCERT/CCも注意喚起を発表した,,。Amazonの脅威インテリジェンスチームは、Earth LamiaやJackpot Pandaを含む中国系の脅威アクターが、このRCE脆弱性の開示を監視し、PoC公開後、他の既知のN-day脆弱性（例：CVE-2025-1338）と**同時に迅速に悪用**していることを観測した,,。この脆弱性の緩和策を適用しようとした際、Cloudflareは設定ミスが発生し、2025年12月5日に約25分間にわたるグローバルなトラフィック障害を引き起こす事態となった,,。

## AIエージェントの新攻撃手法と長期的な企業リスク
セキュリティ研究者により、LLM駆動型ブラウザエージェントを標的とする「Zero-Click Agentic Browser Attack」が発見された,。この攻撃は、エージェントが持つ**「過剰なエージェンシー（Excessive Agency）」**という性質を悪用し,、巧妙に細工された自然言語の指令を埋め込んだメールを介して、ユーザーの確認なしにGoogle Drive内の重要なコンテンツを削除・移動させる,。
また、国内ではアサヒグループホールディングスへのランサムウェア攻撃（2025年9月29日発生）の影響が長期化し、システム復旧後の決算手続きおよび会計監査人による監査手続きの日数を考慮し、**2025年12月期決算発表の延期**という形で企業の財務報告義務とガバナンスに重大な影響が及んでいる,,。

# 2025年12月5日 サイバーセキュリティニュース詳細
## ニュース紹介
### 1. PRC国家アクターによる仮想化基盤標的型マルウェア「BRICKSTORM」の警告
PRC国家支援型アクターは、高度なバックドアマルウェア「BRICKSTORM」を使用して、VMware vSphereおよびvCenterサーバーを標的としている,。これは、従来の防御（ネットワーク境界やエンドポイント）が監視しにくいハイパーバイザーを直接攻撃する**水平展開の究極形態**であり、単なる情報窃取を超え、重要インフラの運用継続性そのものに対する脅威である。このマルウェアは、永続的なアクセスと潜在的な妨害行為を可能にし、主に政府および情報技術（IT）セクターを標的とするが,、アジア太平洋地域の製造業も標的となっている,。CISAは、重要インフラの所有者と運営者に対し、提供されたIOC（侵害の指標）および検出シグネチャを使用して、全てのVMware環境を直ちにスキャンすることを強く要求している,,.
[出典: 2025年12月5日付 サイバー脅威インテリジェンス報告書,, CISA Joint Advisory Warns Critical Infrastructure of BRICKSTORM ...]

### 2. React Server Components RCE脆弱性 (CVE-2025-55182)の緊急悪用
Next.jsのバージョン15.xおよび16.xでApp Routerを利用している環境に影響を与える、React Server ComponentsのRCE脆弱性が報告された,。この脆弱性（React Server Componentsにおける不正なデシリアライズ）はCVSSスコア10.0であり、認証を必要とせず、細工したHTTPリクエストでリモートからコードを実行できる,。PoCの公開後、Amazonの脅威インテリジェンスチームは、Earth LamiaやJackpot Pandaといった中国系アクターが、この脆弱性を他のN-day脆弱性と**同時に迅速に悪用**していることを確認した,,。Cloudflareは、この脆弱性の緩和策を適用しようとした際に設定ミスが発生し、一時的にグローバルなトラフィック障害を引き起こし、クリティカルパッチの適用における迅速性と可用性のジレンマを浮き彫りにした,。
[出典: China-nexus cyber threat groups rapidly exploit React2Shell vulnerability (CVE-2025-55182) | Amazon Web Services,, 「React」脆弱性、実証コード公開 - 悪用リスクが上昇 - Security NEXT, Cloudflare outage on December 5, 2025]

### 3. LLM駆動型エージェントを悪用したZero-Clickデータワイパー攻撃の発見
LLM駆動型システムにおける根本的なセキュリティ欠陥を突く新たな攻撃手法、「Zero-Click Agentic Browser Attack」が発見された,。この攻撃は、GmailやGoogle DriveにOAuthアクセスを持つブラウザエージェントを標的とする,。攻撃者は、自然言語で書かれた指令（例：「Driveのクリーンアップ」）をメールに埋め込み、エージェントの**「過剰なエージェンシー」**を悪用する,。エージェントはこれを正当なルーティン作業と解釈し、ユーザーの確認なしに、Google Drive内の重要なコンテンツをゴミ箱に移動・削除する,。これは、AIの振る舞いと権限設定の脆弱性を突くものであり、攻撃が共有フォルダやチームドライブに迅速に伝播し、大規模なデータ消去を引き起こすリスクがある,。
[出典: Zero-Click Agentic Browser Attack Can Delete Entire Google Drive ...,,]

## 深掘り
### 脅威トレンドの進化とレジリエンスの再定義

#### 長期的なガバナンス・財務リスクの顕在化（アサヒGHD事例）
アサヒグループホールディングスの事例は、大規模なサイバーインシデントの影響がIT領域を超越し、企業の**財務報告義務、ガバナンス、および規制遵守**に直接的な連鎖的影響を及ぼすことを明確に示している。2025年9月29日に発生したランサムウェア攻撃に起因し、同社はシステム復旧後の決算手続きおよび会計監査人による監査手続きにかかる日数を考慮して、2025年12月期決算発表の延期を決定した,。これは、ITリスクが完全に規制・コンプライアンスリスクへ移行したことを意味する。したがって、企業はシステム復旧計画において、従来の平均復旧時間（MTTR）だけでなく、**「Time To Audit Readiness (監査準備完了までの時間)」**を新たな重要KPIとして組み込むことで、レジリエンスの定義を拡張する必要がある。

#### ステルス化するマルウェアとAIエージェントへの対策
現在のマルウェアのエコシステムでは、攻撃者が検知を回避するためにステルス戦術を採用しており、ウォッチガードのレポートによると、**マルウェアの70%**がTLS（トランスポートレイヤセキュリティ）を利用した**暗号化された接続を介して配信**されている,,。これは、従来の非TLS検査に依存する境界防御の可視性を大きく損なう,。
一方、LLM駆動型システムへのZero-Click Agentic Attackの出現は、セキュリティガバナンスをコードレベルから**言語レベル**へ引き上げる必要性を示唆している。対策として、LLMエージェントに与えるOAuth権限を厳格な最小特権原則に基づいて設定し、自律的なファイル削除や機密情報の外部送信といった高リスクなアクションを許可しないよう、設定を強制することが不可欠である,。

### 関連知識

#### 用語解説
*   **BRICKSTORMマルウェア:** PRC国家支援型アクターが使用する高度なバックドアマルウェア,。VMware vSphereおよびvCenterサーバーなどの仮想化基盤を標的とし、永続的なアクセスと潜在的な妨害行為を可能にする戦略的ツールとして分析されている,。
*   **CVE-2025-55182（React2Shell）:** React Server Componentsにおけるunsafe deserialization（不正なデシリアライズ）の脆弱性で、CVSSスコアは最高の10.0,,,。認証なしにリモートコード実行を可能にする。
*   **Agentic Attack（エージェント攻撃）:** LLM駆動型エージェントが持つ「過剰なエージェンシー」を悪用し、悪意のある自然言語の指令を実行させる新しい攻撃手法,。

#### 今後の展望
攻撃者は新しい脆弱性の開示を監視し、PoC公開後、防御側のパッチ適用サイクルを上回る速度で迅速に悪用している,。企業は、クリティカルパッチの適用において、迅速性だけでなく、Cloudflareの事例が示したように、サービス可用性への影響を避けるための厳格な変更管理と影響評価が不可欠となる,。また、中長期的な戦略として、暗号化されたマルウェア配信に対抗するため、TLS/SSLインスペクション技術の導入加速が推奨される。

## 結論
2025年12月5日の脅威環境は、国家主導型脅威が仮想基盤という組織の心臓部を標的とする戦略的変化と、AIエージェントの「振る舞い」を突く新時代の脅威の台頭によって特徴づけられる,。組織は、BRICKSTORM対応のためのIOCスキャンやReact RCE脆弱性への緊急パッチ適用といったP1対応を即座に実施しつつ,、アサヒGHDの事例から得られた教訓に基づき、財務、法務、IR部門を巻き込んだインシデント対応訓練を実施し、**「監査準備完了までの時間」**を指標に加えた総合的なレジリエンスの構築を急ぐ必要がある,。
