# サイバーセキュリティニュースレポート：2025年11月5日

## 1. 概要

本レポートは、過去24時間以内に更新されたサイバーセキュリティ関連のニュースを、日本国内の動向を中心に、海外の重大事例を含めてまとめたものです。本日の主要なトピックは、**日本の大手メディア企業における大規模なデータ漏洩**、**フィッシング攻撃の巧妙化と急増**、そして**高深刻度の脆弱性に関する技術的な警告**の3点です。

## 2. 日本国内のサイバーセキュリティ動向

### 2.1. フィッシング攻撃の巧妙化と増加傾向

日本国内において、フィッシング攻撃が過去5年間で約30倍に急増していることが報告されています [3]。従来の金融機関などを装う手口に加え、**消費者向けウェブサイトを標的とする手口が巧妙化**しており、防御側が後手に回る状況が続いています。

特に脅威となっているのが「**リアルタイムフィッシング**」と呼ばれる手法です。これは、攻撃者が偽のウェブサイトを通じてユーザーの認証情報をリアルタイムで盗み取り、その情報を用いて即座に正規のサービスにログインを試みるものです。この手法は、**二段階認証（MFA）の仕組みをも無力化する可能性**があり、従来のセキュリティ対策の限界を示唆しています。

企業や個人は、フィッシングメールやSMSに記載されたリンクを安易にクリックしない、不審なサイトで認証情報を入力しないといった基本的な対策に加え、MFAの導入状況や、より高度なフィッシング対策技術の検討が急務となっています。

## 3. 海外の重大サイバーセキュリティ事例

### 3.1. 日本の大手メディア企業における大規模データ漏洩

日本の大手メディア企業である日経新聞社が、従業員のSlackアカウントへの不正アクセスによるデータ漏洩の被害を公表しました [1]。このインシデントは、**従業員の個人用コンピューターがインフォスティーラー型マルウェアに感染**したことが原因で、Slackの認証情報が盗まれたことに端を発しています。

盗まれた認証情報は、従業員のSlackアカウントへのアクセスに悪用され、結果として**17,000人以上のSlackユーザー**（従業員およびビジネスパートナー）の**氏名、メールアドレス、およびチャット履歴**が漏洩しました。同社は、情報源や報道活動に関連する情報の漏洩は確認されていないとしていますが、企業内部のコミュニケーションツールが攻撃の足がかりとなった重大な事例として注目されています。

この事例は、企業が提供するデバイスだけでなく、従業員の個人用デバイスのセキュリティ管理の重要性、および認証情報を盗み出すマルウェアの脅威が依然として高いことを示しています。

## 4. 技術的な警告と脆弱性情報

### 4.1. Elastic Cloud Enterpriseの高深刻度脆弱性（CVE-2025-37736）

Elastic Cloud Enterprise（ECE）において、高深刻度（CVSS 8.8/High）の脆弱性「**CVE-2025-37736**」が公表されました [2]。この脆弱性は、**不適切な認可（Improper Authorization）**に起因するもので、組み込みの「readonly」ユーザーが、本来許可されない管理系APIを実行できてしまうというものです。

この脆弱性が悪用されると、readonlyユーザーの権限で**ユーザーの作成・変更・削除、APIキーの投入、サービスアカウントの操作**といった管理権限レベルの操作が可能となり、**実質的な管理者権限の奪取**に至る恐れがあります。

**影響を受けるバージョン**は、ECE 3.8.0以降かつ3.8.2まで、および4.0.0以降かつ4.0.2までです。

**対策**として、Elastic社は直ちに**ECE 3.8.3 または 4.0.3**への更新を強く推奨しています。ECEを利用しているすべてのユーザーは、速やかに対応を行う必要があります。

## 5. 用語解説：インフォスティーラー型マルウェア

本レポートで言及された**インフォスティーラー型マルウェア**（Infostealer Malware）とは、感染したコンピューターから機密情報を盗み出すことを目的とした悪意のあるソフトウェアの総称です。

主に以下のような情報を標的とします。

*   **認証情報:** ブラウザに保存されたパスワード、Cookie、セッション情報、VPNやFTPクライアントの認証情報など。
*   **仮想通貨ウォレット:** 秘密鍵やシードフレーズ。
*   **機密ファイル:** 特定の拡張子を持つドキュメントファイル。
*   **システム情報:** インストールされているソフトウェアやハードウェアの詳細。

インフォスティーラーは、多くの場合、フィッシングメールの添付ファイルや、不正なウェブサイトからのダウンロード、あるいは他のマルウェアの二次感染として拡散されます。盗み出された情報は、攻撃者のコマンド＆コントロール（C2）サーバーに送信され、ダークウェブなどで売買されることが一般的です。
