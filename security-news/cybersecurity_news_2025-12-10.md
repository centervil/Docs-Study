# 2025年12月10日 サイバーセキュリティニュースまとめ
## 主要ポイント
* **Windowsのゼロデイ脆弱性（CVE-2025-62221）** がアクティブに悪用されており、Microsoftは緊急パッチを公開しました。これはローカルアクセスを持つ攻撃者にSYSTEM権限への**権限昇格（EoP）**を許すもので、最優先での対応が推奨されています,,,。
* 韓国のEコマース大手Coupangの大規模データ漏洩事件に関し、警察は2日間にわたり本社を家宅捜索し、**元中国人従業員**を情報通信ネットワーク法違反の容疑で特定しました,,,。この事件は、内部脅威管理の重要性を浮き彫りにしています。
* SSHハニーポットのテストでは、**24時間で20,000件以上**のログイン試行が記録され、主にルーマニアやオランダのボットネットネットワークから発信されており、自動化された攻撃がネットワークエッジに常態的に圧力をかけていることが示されました,,。

## 脆弱性管理とゼロデイ攻撃
Microsoftは12月のPatch Tuesdayで合計57件の脆弱性に対処し、そのうち3件がCritical、53件がImportantに評価されました,。特筆すべきは、**Windows Cloud Files Mini Filter Driver**のUse-after-freeの脆弱性（CVE-2025-62221、CVSSスコア7.8）が、パッチ公開時点で既に**アクティブに悪用されていた**ことです,,,。これはローカルアクセスを持つ攻撃者がSYSTEM権限を獲得するために悪用される可能性があります。また、GitHub Copilot for JetBrains（CVE-2025-64671）とWindows PowerShell（CVE-2025-54100）においても、ローカルでのコード実行を可能にするコマンドインジェクションの脆弱性がゼロデイとして修正されました,。

## 重大インシデントと内部脅威
韓国のCoupangで発生した33.7百万件の顧客情報漏洩事件に関して、ソウル警視庁のサイバー捜査チームは、2日連続でCoupang本社に対する家宅捜索を実施しました,,。捜査の結果、警察はデータ漏洩に使用されたIPアドレスを特定し、容疑者として**元従業員である中国人**を特定しました,,。この事案は、退職者による不正行為が大規模な情報流出の原因となる「内部脅威」が非常に現実的なリスクであることを示しています,。

## 脅威の動向とインシデント事例
公開されたSSHハニーポットのテスト結果から、たった24時間で公開IPアドレス上のポート22に対し**20,000回以上**のログイン試行が記録されました,。攻撃の大部分は、長年にわたり活動しているルーマニアのボットネットネットワークから来ています。一方、攻撃者はユーザーの警戒を回避するため、正規のインフラを悪用する戦術も展開しており、Mimecastの正規リンクを利用してSharePointやDocuSignを偽装したフィッシングメールが40,000通も送信された事例も確認されています,。国内では、大気社の海外グループ会社へのランサムウェア攻撃で社外への情報流出は確認されなかったとの報告があった一方で、日本リウマチ学会運営サイトへの不正アクセスで非公開ページへの不正な書き込みが確認されています。

# 2025年12月10日 サイバーセキュリティニュース詳細
## ニュース紹介
### 1. Windows Cloud Files Mini Filter Driverのゼロデイ脆弱性（CVE-2025-62221）がアクティブに悪用される
Microsoftは2025年12月の月例パッチで、**Windows Cloud Files Mini Filter Driver**に存在する権限昇格の脆弱性（CVE-2025-62221）を修正しました,,。この脆弱性はUse-after-free（解放後メモリ使用）の問題であり、攻撃の複雑性は低いと評価されています,。既にアクティブなゼロデイ攻撃で悪用されており、攻撃者がシステムへのローカルアクセスを得た後、この脆弱性を突いてアクセス権限を昇格させ、最終的に**SYSTEM権限**を取得することが可能になります,。このドライバーはOneDrive、Google Drive、iCloudなどのクラウドストレージアプリケーションによって利用されていますが、これらのアプリがインストールされていないシステムにもコアなWindowsコンポーネントとして存在しています。CISA（米国サイバーセキュリティ・インフラストラクチャセキュリティ庁）は、連邦政府機関に対し2025年12月30日までにパッチ適用を義務付けています。
[出典: CSO - December Patch Tuesday: Windows Cloud Files Mini Filter Driver hole already being exploited],,,,
### 2. Coupangデータ漏洩事件で元従業員の中国人容疑者を特定、警察が本社に2日連続の家宅捜索
韓国のEコマース大手Coupangは、同国人口の半分以上に相当する33.7百万人の顧客情報（氏名、電話番号、メールアドレス、配送詳細など）が漏洩した事案で、警察の捜査を受けています,,。ソウル警視庁のサイバー捜査チームは、証拠を押収するため、10時間にわたる初日の捜索に続き、2日連続でCoupang本社に家宅捜索に入りました,,。警察は、情報通信ネットワーク法違反と機密データ漏洩の容疑で、**Coupangの元従業員である中国人**を容疑者として特定しました,。警察は、データ漏洩に使用されたIPアドレスを確保し、システムにおける潜在的な技術的脆弱性についても調査を進めています,。
[出典: The Korea Times - Police raid Coupang headquarters for 2nd day over massive data breach],,,,
### 3. SSHハニーポットテストで24時間に20,000件の自動ログイン攻撃を記録
セキュリティ専門家が公開IPアドレスのポート22に**Cowrie SSH honeypot**を展開したテストによると、わずか**24時間**で20,000件以上のログイン試行が記録されました,。実際のSSHサービスはTailscaleネットワーク内（ランダムな3xxxxポート）に隠蔽され、外部からはアクセス不能な設定でした。この攻撃のほとんどは、長年にわたりスキャンと攻撃を継続している**ルーマニアのボットネットネットワーク**（侵害されたデバイス）から来ています,。この設定は、実際のサーバーを完全に安全に保ちながら、攻撃者に偽のシステムで時間を浪費させる効果がありました。攻撃の統計、IPの内訳、コマンドログなどは、独自のログ分析ユーティリティ「cowview」を使用して追跡されました。
[出典: Reddit - 20.000 SSH Attacks in 24h from Roamania, Netherland and Germany— Thanks Tailscale (Honeypot Test)],

## 深掘り
### 権限昇格と内部脅威の組み合わせによる「完全な侵害」
最新の脅威動向は、単一の脆弱性ではなく、初期侵入（足場）と権限昇格（EoP）を組み合わせた攻撃連鎖が常態化していることを示しています。

#### 権限昇格（EoP）脆弱性の役割
Windows Cloud Files Mini Filter DriverのEoP脆弱性（CVE-2025-62221）は、ローカルアクセスを完全なシステム支配に転換させる役割を果たします,。Tenableの専門家は、EoPバグは、攻撃者がソーシャルエンジニアリングや他の脆弱性の悪用によって初期アクセスを得た後に、**ポスト・コンプロマイズ（侵害後）活動**を行うために利用されると指摘しています,。これは、防御側がパッチのテストや展開に要する標準的な遅延期間が、攻撃者にシステム乗っ取りを事実上許容する時間的な窓口を提供することを意味します。

#### 内部脅威としてのクロスボーダー・ガバナンスリスク
Coupangのデータ漏洩事件は、外部からのRCE攻撃よりも、退職した特権ユーザーがそのアクセス権限を利用して機密データを持ち出すという、**比較的単純な内部犯行**が甚大な被害を引き起こす可能性を浮き彫りにしました。容疑者が元従業員（中国人）であったという事実は、特に日本企業のような国際的な事業展開を行う組織に対し、海外拠点でのデータアクセス管理と、退職時の**デジタル・オフボーディング**（アクセス権の即時かつ完全な剥奪）の厳格化が急務であることを示唆しています。

### 関連知識
#### 用語解説
*   **Use-after-free (UAF):** プログラムが、既にシステムに返却され（解放され）たメモリブロックを使用しようとするときに発生する脆弱性です,。攻撃者がこの問題を利用してメモリの内容を操作することで、権限昇格などの不正な動作を引き起こすことができます。
*   **Windows Cloud Files Mini Filter Driver:** クラウドストレージアプリケーション（OneDrive, Google Driveなど）がファイルシステム機能にアクセスできるようにするために利用される、Windowsのファイルシステムフィルタードライバー（ミニフィルター）です,。

#### 今後の展望
今後注視すべきは、Microsoftが修正したCVE-2025-62221に対するPoC（Proof-of-Concept）エクスプロイトコードが公に公開されるか否かです。PoCが公開されれば、攻撃の規模は劇的に拡大すると予測されます。また、AIの能力向上は、攻撃側が「信頼性の高いフィッシング文面」や巧妙なクリック誘導手法を効率的に作り出すことを可能にしており,、防御側は、境界防御の強化だけでなく、異常な振る舞いの迅速な**検知（Detection）とレスポンス（Response）の自動化**技術の導入を急ぐ必要があります,。

## 結論
現在、企業は「アクティブに悪用されているゼロデイ」という緊急の技術的脅威（CVE-2025-62221）と、「元従業員による大規模データ漏洩」という深刻な内部脅威（Coupang事例）という二重のリスクに直面しています。セキュリティリーダーは、単なるCVSSスコアに依存せず、**実際の悪用状況に基づいた即時対応パラダイム**を採用し、Windowsパッチを最優先で展開する必要があります,。同時に、Coupangの教訓に基づき、特権アクセスログの緊急監査を実施し、退職者を含む**内部脅威対策と国際的なガバナンス**を強化することが、事業継続性を確保するための必須事項です,,。
