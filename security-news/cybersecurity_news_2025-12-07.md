# 2025年12月7日 サイバーセキュリティニュースまとめ
## 主要ポイント
* **CVSSスコア10.0のRCE脆弱性React2Shell（CVE-2025-55182）**が、公表から**数時間以内**にEarth LamiaやJackpot Pandaを含む複数の中国系APTグループによって活発に悪用されている事実が観測された。
* 元CSIS長官は、敵対勢力によるスパイ活動の「最前線」が、政府機関から**民間セクターのイノベーション、研究開発、および大学**へと移行し、「産業規模」の技術盗用が行われていると警告した。
* 脅威インテリジェンス（TI）への投資は増加傾向にあり、91%の組織が2026年に支出を増額する計画だが、**「統合の質の低さ」**が最大の運用課題となっている。

## 緊急脆弱性（React2Shell）とAPT活動
最新のインテリジェンス報告は、モダンなWebアプリケーション開発で広く使用されるReactおよびNext.jsに影響を与える極めて深刻な脆弱性、CVE-2025-55182（React2Shell）が、既に現実世界の脅威となっていることを示しています。この脆弱性は、React Server Componentsにおける危険なデシリアライゼーションの欠陥であり、CVSSスコアが最大の**10.0**です。Amazonの脅威インテリジェンスチームは、公開（2025年12月3日）から**数時間以内**に、Earth LamiaやJackpot Pandaを含む複数の中国系国家関連APTグループによる活発な悪用試行を確認しました。この極端な悪用速度は、従来のパッチ管理における対応猶予時間（Time-to-Exploit）が完全に失われたことを示唆しており、防御体制は高度に自動化された継続的な脆弱性管理へとシフトせざるを得ません。攻撃者は、自動スキャンツールや、技術的に不完全な公開PoC（Proof-of-Concept）でさえも利用し、広範なキャンペーンを展開することで、脆弱なターゲットを見つける成功率を最大化しようとしています。

## 地政学的リスクと産業スパイ活動の深化
グローバルな脅威環境は、国家間の軍事技術優位性をめぐる産業的・学術的な破壊工作へとその性質を変化させています。元カナダ情報サービス（CSIS）長官のDavid Vigneault氏は、敵対的なスパイ活動の焦点が、政府から**民間部門のイノベーション、研究イノベーション、および大学**へと劇的に転換したと警告しました。これは、中国が主導する「産業規模」の技術盗用プログラムの一部であり、サイバー攻撃、潜入工作員、および大学職員の採用を組み合わせることで、盗み出した技術を中国人民解放軍（PLA）の生産に投入することが最終目的です。
さらに、フランスのエマニュエル・マクロン大統領は、中国の産業の台頭が、ヨーロッパの産業モデル（工作機械や自動車）の「中心部に衝突している」と警告しました。中国の貿易黒字が3,000億ユーロ（3,495億ドル）を超えた持続不可能なレベルに達していることを指摘し、是正されなければ、EUが米国と同様に、関税などの**デカップリング措置**を講じる可能性があると示唆しました。

## 脅威インテリジェンス（TI）の運用と投資の動向
脅威インテリジェンス（TI）は、国家支援攻撃やサイバー犯罪の増加に対応するため、「ミッションクリティカル」な機能として確立されています。Recorded Futureの年次報告書によると、セキュリティ意思決定者の43%がTIを戦略的な計画やビジネス投資の指針として活用しており、データ侵害が増加する中、91%の組織が2026年にTIへの支出を増額する計画です。
しかし、投資が増えているにもかかわらず、半数以上の組織が自らのTIプログラムを「未成熟」と認識しており、「統合の質の低さ」が最大の運用課題となっています。この運用上の成熟度ギャップに対処し、TIの効率を高めるため、セキュリティプロフェッショナル全体の81%が、今後12か月間でTIベンダーの**統合（consolidation）**を加速する計画です。

# 2025年12月7日 サイバーセキュリティニュース詳細
## ニュース紹介
### 1. React2Shell脆弱性（CVE-2025-55182）の即時悪用
React Server Componentsの深刻な脆弱性であるCVE-2025-55182（React2Shell）は、CVSSスコア10.0の未認証リモートコード実行（RCE）を可能にする欠陥です。この脆弱性は、Reactバージョン19.xおよびApp Routerを使用しているNext.jsバージョン15.x/16.xに影響します。公開後わずか数時間で、Earth LamiaやJackpot Pandaなどの中国系APTグループによる悪用が観測されました。これらの脅威アクターは、CVE-2025-55182の悪用と並行して、他のN-day脆弱性（例: CVE-2025-1338）も同時に悪用する体系的なアプローチを採用しています。AWSは、この脆弱性はAWSサービスには影響しないとしつつも、EC2やコンテナなどの独自の環境でReactまたはNext.jsアプリケーションを実行している顧客に対し、**直ちに脆弱なアプリケーションを更新する**よう強く推奨しています。
[出典: China-nexus cyber threat groups rapidly exploit React2Shell vulnerability (CVE-2025-55182) | Amazon Web Services]

### 2. カナダ元情報長官による産業スパイ活動の警告
元カナダ情報サービス（CSIS）長官のDavid Vigneault氏は、敵対勢力によるスパイ活動の最前線が、政府情報から**民間部門のイノベーション、研究イノベーション、大学**へと移行したと警告しました。Vigneault氏によると、中国は、サイバー攻撃、潜入工作員、および大学職員の採用（無知、イデオロギー、または金銭的誘惑に基づく）を組み合わせた「産業規模」での技術盗用を行っています。この目的は、盗み出した技術から軍事的な応用を抽出し、中国人民解放軍（PLA）の生産に投入することで、非対称的な能力に投資することです。同氏は、研究者に対して、セキュリティ規則が厳しすぎるとの批判を退け、「孤島で人類のために純粋な研究を行っていると想像することはできない」と述べ、社会全体でこの脅威と戦う必要があることを強調しました。
[出典: Hostile powers sending spies to west's universities, says former security chief]

### 3. 審調社ランサムウェア被害の最終報告とサプライチェーンリスク
調査業務を手がける株式会社審調社は、2025年7月に発生した自社ネットワークへの不正アクセスおよびランサムウェア被害について、被害内容と原因、再発防止策を取りまとめた最終報告を2025年12月5日に公表しました。このインシデントでは、ファイルの一部がランサムウェアにより暗号化されるとともに、外部に窃取される被害が発生しました。漏洩またはそのおそれがある情報には、委託元から受託した調査データが含まれ、特に「番号と氏名等および要配慮個人情報（医療情報）が含まれるもの」が約1,200件確認されています。漏洩した可能性のある要配慮個人情報は医療情報のみであり、マイナンバーやクレジットカード情報など財産的被害に直結し得る情報は同社では保有していなかったとされています。この事案は、保険会社や共済などの**委託元の顧客情報が委託先を通じて影響を受けるサプライチェーン型インシデント**の一例であり、保険・金融分野における委託先を含めた一体的なセキュリティガバナンスの構築の重要性が改めて示されました。
[出典: 審調社、ランサムウェアの被害で個人情報漏洩の恐れ|セキュリティニュース]

## 深掘り
### Time-to-Exploitの圧縮と防御戦略の転換
React2Shellの事例は、Criticalな脆弱性が公開から**数時間**で国家関連アクターによって悪用されるという、現代のサイバー脅威環境における「Time-to-Exploit」の極端な圧縮を明確に示しています。従来のセキュリティポリシーでCVSS 10.0の脆弱性に対して設定されていた72時間や7日間といった対応期間は、APTの武器化速度が数時間に圧縮されたことで通用しなくなりました。したがって、防御側は、手動による介入から、高度に**自動化された継続的な脆弱性管理（Continuous Vulnerability Management）**へと防御体制をシフトさせることが必須となります。攻撃者は、たとえ技術的に不完全な公開PoCであっても、それを迅速にスキャンインフラストラクチャに統合し、広範なキャンペーンを展開することで成功率を最大化しようとしています（Volume-based approach）。

#### 攻撃者の悪用手法と防御策
攻撃者は、ユーザーエージェントのランダム化といった検出回避機能を持つ自動スキャンツールや、個別のPoCエクスプロイトを使用して活動しています。観察された攻撃活動の指標には、アプリケーションエンドポイントへの`next-action`または`rsc-action-id`ヘッダーを含むHTTP POSTリクエストや、偵察コマンド（`whoami`, `id`）の予期せぬ実行試行、`/etc/passwd`の読み取り試行などが含まれます。
AWSは、Sonaris脅威インテリジェンスシステムによる悪意あるスキャン試行の自動検出・制限や、AWS WAFの`AWSManagedRulesKnownBadInputsRuleSet`（バージョン1.24以上）への更新されたルールの追加により、顧客保護を支援しています。しかし、これらの保護はパッチ適用を代替するものではないため、顧客自身が脆弱なアプリケーションを更新することが最も重要です。

#### 脅威インテリジェンス（TI）の「運用化負債」
脅威インテリジェンスに対する信頼と投資が増加している一方で、その運用上の成熟度にはギャップが存在します。半数以上の組織がTIプログラムを「未成熟」と見なす最大の原因は、ほぼ半数の回答者が指摘する**「統合の質の低さ（poor integration）」**です。TIデータが既存のセキュリティ運用ツール（SIEM, SOAR, EDR）に効果的に組み込まれていない場合、多額のTI投資は「運用化負債」を生み出します。TIの価値を最大化するためには、予算を「データソースの購入」から「データのアクション化」へとシフトさせ、TIベンダーの統合を加速し、取得したインテリジェンスが自動的に修復措置をトリガーするように運用を改善することが推奨されています。

### 関連知識
#### 用語解説
*   **React2Shell (CVE-2025-55182)**: React Server Componentsにおける安全でないデシリアライゼーションの脆弱性であり、CVSSスコア10.0の最大深刻度の未認証リモートコード実行（RCE）を可能にします。
*   **Earth Lamia / Jackpot Panda**: React2Shellの悪用を迅速に行ったことが確認された、中国系の国家関連サイバー脅威アクター（APTグループ）です。Earth Lamiaは金融サービス、物流、IT企業、大学などを標的にし、Jackpot Pandaは主に東アジアおよび東南アジアのエンティティを標的にしています。
*   **多要素認証（MFA）/ パスキー**: 不正ログインを防ぐための効果的な対策の一つです。多要素認証は、知識情報、所持情報、生体情報のうち2つ以上を組み合わせる認証であり、パスキーはパスワードを利用しない認証方式です。

#### 今後の展望
**技術的対応**として、Critical RCE脆弱性に対するインシデント対応プロトコルを、APTの武器化速度（数時間）に合わせて改定し、インターネット公開資産に対するパッチ展開システムの自動化機能を強化することが急務です。
**戦略的防御**として、CSISが警告する産業スパイ活動に対抗するため、イノベーション部門や大学との共同研究ネットワーク（R&D資産）を厳格に分離し（マイクロセグメンテーション）、サイバー防御に加え、人的要素（HUMINT）リスクを低減するためのプロトコル（機密情報アクセス権を持つ職員への教育や行動監視）を導入することが不可欠です。
**地政学的リスク**においては、マクロン大統領の警告が示すように、EUが中国に対して保護主義的措置（関税など）を講じる可能性があり、日本企業はサプライチェーンや市場アクセスに対する規制強化の影響を脅威インテリジェンスとして分析し、経営層に報告する体制を確立する必要があります。

## 結論
2025年12月7日現在のサイバー脅威環境は、技術的な脅威の速度（React2Shellの即時悪用）が防御側の対応能力を上回り、同時に国家主導のIP盗用プログラムや地政学的貿易摩擦の深化という戦略的な課題が表面化していることを示しています。企業は、RCE脆弱性に対する即時封じ込めを自動化するための運用改善に投資しつつ、知的財産保護のために人的リスク対策を組み込んだ全方位的な防御を確立することが、長期的な存続のために不可欠です。また、基本的な防御策（多要素認証や情報リテラシーの向上）の徹底は、高度な脅威に対する初期侵入経路を断つための基礎として引き続き重要です。
