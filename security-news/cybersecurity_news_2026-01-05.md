# 2026年1月5日 サイバーセキュリティニュースまとめ

## 主要ポイント
* **台湾の重要インフラを狙う中国のサイバー攻撃が急増：** 2025年の1日平均侵入試行数は263万回に達し、特にエネルギー部門への攻撃は前年比10倍（1,000%増）という驚異的な伸びを記録しました。
* **IBM API Connectにおける致命的な認証バイパス脆弱性：** リモートから認証を回避可能な極めて深刻な脆弱性（CVE-2025-13915、CVSS 9.8）が公表され、管理下にあるバックエンドサービスや機密データが完全に侵害されるリスクが生じています。
* **「災害級」と評される日本のサイバー脅威：** JNSAが発表した「2025セキュリティ十大ニュース」では、アサヒGHDやアスクルを襲ったランサムウェア攻撃が1位となり、サプライチェーン全体を麻痺させる脅威が浮き彫りとなりました。

## 重要インフラおよび国家背景の脅威
台湾国家安全局（NSB）の報告によれば、中国のサイバー軍は脆弱性の悪用、DDoS、ソーシャルエンジニアリング、サプライチェーン攻撃という4つの主要戦術を駆使し、エネルギーや金融など9つの主要セクターを執拗に標的にしています。また、ルーマニアでも年末年始の休暇期間を狙い、水管理当局や国内最大の石炭火力発電事業者が相次いでサイバー攻撃を受け、事務インフラが麻痺する事案が発生しました。

## 企業・団体のデータ侵害と国内動向
国内では、TOKYO FMが元日に300万件超のデータ流出疑惑（検証保留中）に直面したほか、ステッカージャパンでも約11万人分の個人情報がダークウェブで販売されるなどの事案が報告されています。JNSAのランキングでは、証券口座の乗っ取り被害の急増（3位）や能動的サイバー防御の関連法案成立（5位）など、官民両面での大きな変化が注目されました。

## 脆弱性対策と国際捜査の成果
IBMの脆弱性に加え、MongoDBの「MongoBleed」脆弱性の悪用も世界的に観測されています。一方で、インターポール主導の「Operation Sentinel」により、トレンドマイクロらの協力のもと、アフリカ全土でランサムウェアやビジネスメール詐欺に関与した574人が摘発されるなど、国際的な法執行機関の取り組みも成果を上げています。

# 2026年1月5日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. 台湾へのサイバー攻撃、エネルギー部門で10倍に増加
台湾国家安全局（NSB）は、2025年の中国による台湾重要インフラへのサイバー攻撃分析レポートを公開しました。1日平均の侵入試行は263万回と、2023年の123万回から倍増しています。攻撃の57%はハードウェアやソフトウェアの脆弱性を悪用したもので、BlackTechやMustang Pandaといった主要なハッカーグループの関与が特定されました。
[出典: Taipei Times - Chinese cyberattacks rising: NSB report](https://www.taipeitimes.com/News/front/archives/2026/01/05/2003847426)

### 2. IBM API Connectに認証バイパスの脆弱性（CVE-2025-13915）
IBMは、API管理プラットフォーム「IBM API Connect」に、リモートから認証チェックを完全に回避できる重大な脆弱性が存在することを発表しました。この脆弱性はユーザーの操作を必要とせず、悪用されると機密データへの不正アクセスやシステムの完全な制御を許す可能性があります。バージョン10.x系が広く影響を受けており、IBMは即時の修正パッチ適用を強く推奨しています。
[出典: The Cyber Express - Critical IBM API Connect Vulnerability Enables Authentication Bypass](https://thecyberexpress.com/ibm-api-connect-vulnerability-auth-bypass/)

### 3. ルーマニアの重要インフラ、休暇中に組織的サイバー攻撃を相次ぎ受ける
2025年12月末、ルーマニアの国立水資源管理当局と、国内電力の30%を供給するオルテニア・エネルギー・コンプレックスが相次いでサイバー攻撃を受けました。オルテニア社を襲ったのは「Gentlemen」と呼ばれるランサムウェアグループで、ERPシステムやメール、Webサイトを暗号化し、事務機能を麻痺させました。水管理当局への攻撃では、Windowsの標準機能であるBitLockerが悪用される「Living-off-the-land（環境寄生型）」の手法が取られました。
[出典: Industrial Cyber - Romanian water authority, energy producer hit by cyber attacks in apparent coordinated holiday campaign](https://industrialcyber.co/attacks-and-vulnerabilities/romanian-water-authority-energy-producer-hit-by-cyber-attacks-in-apparent-coordinated-holiday-campaign/)

## 深掘り

### 休暇期間を狙う「タクティカル・ストライク」の脅威
年末年始やクリスマスなどの休暇期間は、セキュリティチームの体制が手薄になり、対応が遅れがちになるため、攻撃者にとって格好の機会となっています。

#### 組織的な波状攻撃とインフラ依存関係の悪用
ルーマニアの事例では、水管理当局がまず攻撃され、その数日後にエネルギー企業が標的となりました。これは、発電所の冷却や水力発電に不可欠なダムの運用を管理する水資源インフラを先に侵害し、依存関係をマッピングした上での戦略的な攻撃であった可能性が指摘されています。また、Adobe ColdFusionを狙った数千件の攻撃もクリスマスの休暇期間に集中して観測されました。

#### 「検証保留中」の情報の扱いと予防的対応
TOKYO FMの事案では、元日に300万件超のデータ流出が主張されましたが、現時点では「検証保留中（Pending Verification）」のステータスにあります。これは虚偽の主張で注目を集める攻撃者もいるため、専門家がデータの真正性を分析している段階ですが、企業側には公式発表前であってもパスワードの変更を促すなどの予防的措置が求められています。

### 関連知識

#### 用語解説
*   **Gentlemen Group:** 2025年に台頭した新興のランサムウェアグループ。非アフィリエイトの密に統制されたチームで、産業組織を集中的に狙うのが特徴。
*   **認証バイパス (Authentication Bypass):** ログイン等の正規の認証プロセスを経由せずに、システムやデータにアクセスできてしまう脆弱性。
*   **能動的サイバー防御 (Active Cyber Defense):** 深刻な被害が懸念される際、事前に攻撃者のサーバーに侵入して無害化するなど、先手を取る防衛手法。日本でも関連法案の成立が注目されました。

#### 今後の展望
NICTの分析によれば、2025年のサイバー攻撃通信件数は過去最多を更新する勢いにあります。今後は技術的な防御に加え、攻撃を受けた際に「いかに早く復旧するか」というレジリエンスの確保や、サプライチェーン全体でのリスク管理が不可欠となります。また、多要素認証（MFA）が必須化される流れにありますが、最近ではそれ自体を突破する手法も増えており、より強固な認証方式（FIDO2等）への移行が加速するでしょう。

## 結論
2025年から2026年にかけての脅威情勢は、単なる情報の窃取を超え、国家インフラや企業の事業継続を直接脅かす「災害級」の事態へと深化しています。特にサプライチェーンの弱点や休暇中の油断を突く組織的な攻撃が顕著であり、官民一体となった情報共有と、被害発生を前提とした事後対応計画（インシデントレスポンス）の徹底が急務です。
