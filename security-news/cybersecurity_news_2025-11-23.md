# 2025年11月23日 サイバーセキュリティニュースまとめ

## 主要ポイント
*   **クラウドインフラストラクチャの信頼崩壊:** Microsoft Azure Bastionにおいて、CVSSスコア10.0の認証バイパス/リモート権限昇格脆弱性（CVE-2025-49752）が報告され、即時パッチ適用が求められています。
*   **金融セクターのサプライチェーン危機:** 不動産サービス技術ベンダーSitusAMCへのサイバー攻撃により、JPモルガン・チェースやモルガン・スタンレーなどの主要銀行の顧客データ（住宅ローン関連の機密個人情報）が漏洩した可能性があり、サードパーティリスクの深刻さが浮き彫りになりました。
*   **サイバーと物理の戦争融合:** イラン関連のアクターが、船舶のAISデータへのデジタル偵察を、物理的なミサイル攻撃の試み（サイバー・イネーブルド・キネティック・ターゲティング）に利用していた事例がAmazonの脅威インテリジェンスにより報告されました。
## 脆弱性と緊急パッチ情報
Microsoft Azure BastionのCVSS 10.0の致命的な脆弱性（CVE-2025-49752）が緊急警告されており、ネットワークベースで悪用可能で、事前の認証やユーザー操作が不要なため、クラウド環境のセキュリティ信頼モデルを崩壊させる重大な欠陥と見なされています。Google Chromeブラウザでは、V8 JavaScriptエンジンにおけるType Confusionゼロデイ脆弱性（CVE-2025-13223、CVSS 8.8）が**積極的に悪用されている**ことが報告され、即座のアップデートが強く推奨されています。また、Fortinet FortiWebのOSコマンドインジェクション脆弱性（CVE-2025-58034）も**積極的に悪用されている**ことが確認され、CISAのKEVカタログに追加されており、連邦機関に対しては1週間の短縮された修正期限が設定されています。Microsoftの2025年11月Patch Tuesdayでは合計63のCVEに対処し、CVSS 9.8のMicrosoft Graphics Component (GDI+) のリモートコード実行 (RCE) 脆弱性（CVE-2025-60724）などが含まれています。
## 大規模侵害とサプライチェーン・リスク
米国の主要金融機関の顧客データが、不動産ローン技術ベンダーであるSitusAMCへのサイバー攻撃により影響を受けた可能性が浮上し、FBIが捜査を開始しました。侵害されたデータは、住宅ローン組成やサービス提供に関連しており、社会保障番号、銀行口座詳細、ローン申請書などの高度に機密性の高い個人財務情報（PII）が含まれる懸念があります。この事件は、金融セクターがニッチだがクリティカルな機能を持つサードパーティベンダーに過度に依存しているという構造的な脆弱性を浮き彫りにしています。
## 地政学的脅威と新たな戦争形態
Amazonの脅威インテリジェンスチームは、イラン関連のアクター（Imperial Kitten）が「サイバー・イネーブルド・キネティック・ターゲティング」という新たな戦争形態を実践していることを明らかにしました。このアクターは、船舶の自動識別システム（AIS）プラットフォームを侵害し、特定の輸送船の追跡情報を収集した後、その数日後にイランが支援するフーシ派によるミサイル攻撃が試みられました。これは、デジタル作戦が物理的な軍事目標を支援するために設計された**連携されたキャンペーン**であり、従来のサイバー防御モデルの限界を露呈しています。

# 2025年11月23日 サイバーセキュリティニュース詳細
## ニュース紹介
### 1. Microsoft Azure Bastion 認証バイパスの緊急警告
Microsoft Azure Bastionに、CVSSスコア**10.0 (CRITICAL)**と評価される致命的な認証バイパスおよびリモート権限昇格脆弱性（CVE-2025-49752）が報告されました。この脆弱性は、認証トークンの不適切な処理（CWE-294）に起因し、攻撃者は事前の認証、特権、またはユーザー操作を一切必要とせず、**リモートから管理者レベルの権限昇格**を達成できます。この欠陥は、仮想マシンへの安全なアクセスを提供するゲートウェイの信頼性を根底から揺るがすものであり、攻撃者は単一のネットワークリクエストを通じてBastionインフラ全体を危険に晒す可能性があります。
[出典: Cyber Security News - Computer Security | Hacking News | Cyber ... - Critical Vulnerability in Azure Bastion Let Attackers Bypass Authentication and Escalate privileges](https://cybersecuritynews.com/)
### 2. SitusAMCサプライチェーン攻撃による米金融機関の顧客データ危機
不動産ローンおよびモーゲージサービス技術の主要ベンダーであるSitusAMCが11月12日にサイバー攻撃を受け、JPモルガン・チェース、シティグループ、モルガン・スタンレーなどの主要銀行の顧客データが影響を受けた可能性が通知されました。SitusAMCは、攻撃が暗号化マルウェアを使用していないこと、および封じ込められたことを発表していますが、漏洩したデータには住宅ローン関連の機密個人情報（社会保障番号、銀行口座詳細、ローン申請書など）が含まれる懸念があり、ID詐欺のリスクが大幅に増加するとされています。FBIが捜査を主導しており、金融セクター全体で緊急の内部リスクレビューが促されています。

### 3. イラン関連アクターによるサイバー・キネティック・ターゲティングの実行
Amazonの脅威インテリジェンスチームは、イランのイスラム革命防衛隊（IRGC）と関連があるとされるImperial Kitten (aka Tortoiseshell)が、デジタル偵察を物理的な軍事作戦に利用する「サイバー・イネーブルド・キネティック・ターゲティング」を行っている事例を特定しました。このアクターは、2024年1月27日に特定の船舶のAIS（自動識別システム）位置データを標的とした検索を実施し、その数日後（2月1日）に、その**まさに同じ船舶**に対してフーシ派によるミサイル攻撃が試みられたことが確認されています。これは、サイバー作戦が、海事インフラに対する物理的攻撃のための正確なインテリジェンスを提供する役割を果たしていることを示しています。

## 深掘り
### 信頼破壊型攻撃とセキュリティ境界の溶解
現代のサイバー脅威は、セキュリティ境界をデジタル領域から物理領域へと拡大する**信頼破壊型攻撃**へと進化しており、企業が信頼して依存している外部要素（クラウドゲートウェイや専門ベンダー）を標的とする傾向が顕著です。

#### クラウド管理レイヤーの致命的脆弱性
Azure BastionのCVE-2025-49752（CVSS 10.0）は、このクラウド管理層を標的とする攻撃が、データセンター全体のセキュリティを瞬時に無効化する能力を持つことを示しており、**高爆発性**リスクとして評価すべきです。この脆弱性は、リモートアクセス、管理、およびゼロトラストアーキテクチャの基盤として利用されるクラウド管理ツールが、致命的な脆弱性によって根本から覆される可能性があることを示しています。

#### サードパーティ依存が引き起こす構造的リスク
SitusAMC事件は、金融業界のシステム的な脆弱性を浮き彫りにしました。大手銀行が自社の防御に自信を持っていても、攻撃者は、最も防御が手薄なリンクであるサードパーティベンダーを体系的に探して攻撃します。米国の規制当局（OCC）は、外部サービスプロバイダーへの過度な依存が**システム的なリスク**を生み出すことを繰り返し警告しており、この事件は防御側の企業のセキュリティ成熟度がサプライチェーン全体によって決定されるという現実を裏付けています。

### 関連知識
#### 用語解説
*   **サイバー・イネーブルド・キネティック・ターゲティング (Cyber-enabled Kinetic Targeting):** 国家アクターがサイバー作戦を体系的に使用し、物理的な軍事作戦を可能にし、強化するキャンペーンを指す、Amazonが提唱する新しい用語です。
*   **CVSS (Common Vulnerability Scoring System):** ソフトウェア脆弱性の深刻度を、0.0から10.0の基本スコアで評価し、その特性を伝えるために使用される無料かつ公開された業界標準です。
*   **KEV (Known Exploited Vulnerabilities) Catalog:** CISA（米国土安全保障省サイバーセキュリティ・インフラセキュリティ庁）が管理する、能動的に悪用が確認されている脆弱性のリストであり、連邦機関に対してはこのリストの脆弱性を期限内（通常は1週間）に修復することが義務付けられています。

#### 今後の展望
サイバー脅威は従来のITシステム保護の範疇を超え、物理的な安全保障に直接影響を与えるパラダイムシフトが進行しています。日本の重要インフラ企業や海運・物流部門は、ITインフラの保護に加えて、OT（運用技術）システムのセンサーデータや地理空間情報とサイバー脅威インテリジェンスを統合して分析する能力を早急に開発し、従来のサイバーセキュリティと物理セキュリティを分離して扱う戦略を転換する必要があります。一方、EUでは、ENISAがCVEプログラムのRootに指定され、EUの脆弱性データベースの開発支援や、サイバーレジリエンス法（CRA）に基づくガイダンス提供を強化することで、国境を越えた脆弱性管理体制の効率化を図っています。

## 結論
本日確認されたAzure BastionのCVSS 10.0脆弱性や、金融セクターを襲ったSitusAMC事件は、企業が依存する技術基盤とサプライチェーンの信頼性が、防御のボトルネックとなっていることを示しています。さらに、イラン関連アクターによる「サイバー・キネティック・ターゲティング」の明確な証拠は、サイバー脅威が国家間の紛争においてリアルタイムの物理的攻撃支援ツールとして機能し始めていることを示唆しており、日本の重要インフラ部門にとっては戦略的な防衛パラダイムの転換が不可欠です。防御側は、アクティブに悪用されているゼロデイへの**即時パッチ適用**と、サプライチェーンリスクの厳格な再評価を最優先事項として実行する必要があります。
