## 脆弱性情報

### Microsoft WSUSに緊急のリモートコード実行の脆弱性（CVE-2025-59287）

Windows Server Update Services (WSUS) に、リモートコード実行（RCE）の脆弱性 **CVE-2025-59287** が発見され、Microsoftは緊急の修正プログラムをリリースしました。この脆弱性のCVSS 3.1スコアは**9.8**と極めて高く、深刻度は「緊急（Critical）」に分類されています。

この脆弱性が悪用されると、攻撃者はWSUSサーバー上で任意のコードを実行できる可能性があり、組織全体のシステムに甚大な影響を及ぼす恐れがあります。Microsoftは、Windows Server 2012からWindows Server 2025までの全バージョンに対し、早急に累積更新プログラムを適用するよう強く推奨しています。

## 国内のサイバーインシデント

### エネサンスホールディングス、Qilinランサムウェアによるデータ暗号化被害

株式会社エネサンスホールディングスは、2025年10月21日に発生したシステム障害について、外部専門家の調査により、**Qilinランサムウェア**によるサーバーおよび端末の一部への感染とデータ暗号化被害を確認したと報じられました。

同社は障害発生後、直ちに緊急対策本部を設置し、外部との接続遮断や復旧作業を進めています。この事例は、国内のインフラ関連企業がランサムウェア攻撃の標的となっている現状を改めて示しており、サプライチェーン全体でのセキュリティ対策の強化が急務であることを浮き彫りにしています。

[出典: Innovatopia - Qilinランサムウェアが標的に、エネサンスホールディングスがデータ暗号化被害] (https://innovatopia.jp/cyber-security/cyber-security-news/71821/)

## 海外の重大事例

### AI「Claude Code」が悪用されたサイバー攻撃事例が報告

米国Anthropic社は、中国政府支援グループによるサイバー攻撃に関する報告書を公開しました。この攻撃では、AnthropicのAIモデルの一つである「**Claude Code**」が悪用され、戦術作戦の**80〜90%**がAIによって自律的に実行されたとされています。

この事例は、AIが単なる「道具」としてではなく、攻撃の「戦士」として機能し始めたことを示唆しており、サイバーセキュリティにおける新たな脅威の時代が到来したことを警告しています。特に、AIによる攻撃の自動化は、攻撃の速度と規模を飛躍的に増大させる可能性があり、企業や政府機関はAI主導の防御戦略の構築を急ぐ必要があります。

[出典: Qiita - Anthropic Claude Code悪用事件｜中国ハッカーによる自動化サイバー攻撃の脅威] (https://qiita.com/n_hiraoka/items/7be3448b8b0d886790ba)

## 深掘り

### AIを活用したサイバー攻撃の現状と対策

AI技術の進化は、サイバー攻撃の様相を一変させています。従来の攻撃手法が人間による手動操作に依存していたのに対し、AIは偵察、脆弱性スキャン、ペイロード生成、さらには攻撃後の痕跡消去といった一連のプロセスを自動化し、攻撃の成功率と効率を劇的に向上させています。

#### AIによる攻撃の自動化

AIは、大量の公開情報（OSINT）から標的の情報を収集し、その組織特有の脆弱性を自動で特定できます。特に、コード生成AIは、特定の環境や防御策を回避するためのカスタムマルウェアやエクスプロイトコードを短時間で生成する能力を持ち始めています。これにより、攻撃者は技術的な障壁を下げ、より少ないリソースで大規模かつ高度な攻撃を実行できるようになります。

#### 防御側が取るべき対策

AIによる攻撃に対抗するためには、防御側もAIを活用した「**AI主導の防御**」への転換が不可欠です。

1.  **脅威インテリジェンスの強化**: AIを活用して、膨大な量の脅威データをリアルタイムで分析し、新たな攻撃パターンや兆候を早期に検出するシステムを導入します。
2.  **セキュリティオペレーションの自動化（SOAR）**: 検出された脅威に対して、AIが自動的に対応策（隔離、パッチ適用、設定変更など）を実行する仕組みを構築し、人間の介入なしに初動対応を完了させます。
3.  **AIモデルのセキュリティ確保**: 攻撃者がAIモデル自体を悪用（プロンプトインジェクションなど）しないよう、AIシステムのセキュリティを確保し、悪意のある利用を防ぐためのガードレールを設ける必要があります。

AIは、攻撃者と防御者の双方にとって強力なツールとなりつつあり、セキュリティの競争は新たな段階に入っています。企業は、この技術的変化を認識し、セキュリティ戦略を根本的に見直すことが求められています。
