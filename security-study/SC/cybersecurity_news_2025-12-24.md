# 2025年12月24日 サイバーセキュリティニュースまとめ

## 主要ポイント
* **国内ネットワーク機器への標的型攻撃：** Array Networks製のAGシリーズにおいて、2025年8月以降、Webshellの設置や内部侵入を伴う悪用被害が国内で発生しているため、緊急の調査と対策が求められている,。
* **国際的なサイバー犯罪掃討作戦の成果：** インターポール主導の「Operation Sentinel」により世界で574人が逮捕されたほか、FBIが銀行の認証情報を窃取する犯罪インフラのドメインを押収するなど、法執行機関による大規模な摘発が相次いでいる,。
* **サプライチェーンとインフラの麻痺：** フランスの郵便・銀行サービスが大規模な攻撃でオフラインになったほか、日産自動車が委託先（Red Hat）の侵害により数万件の顧客情報を漏洩させるなど、外部組織に起因する被害が目立っている,。

## 脆弱性とマルウェアの動向
マイクロソフトは2025年11月の更新プログラムで、SYSTEM特権への昇格が可能な**Windowsカーネルの脆弱性（CVE-2025-62215）**を含む複数の修正を公開した。一方、GitHub上では、最新の脆弱性（CVE-2025-59295等）の**偽のPoC（実証コード）**を装って**WebRAT**マルウェアを配布するキャンペーンが確認された,。この攻撃にはAIが生成したとみられる説明文が使用されており、セキュリティ担当者や開発者を巧みに誘い込む手法がとられている。また、MongoDBにおいても、認証なしでメモリを読み取られる可能性のある高深刻度の脆弱性が発見され、即時のパッチ適用が推奨されている,。

## 国際的な法執行活動
インターポールが主導した**「Operation Sentinel」**は、アフリカを中心に19カ国が参加し、ビジネスメール詐欺やランサムウェアに関与した574人を逮捕、6,000件以上の悪質なリンクを排除する成果を上げた。米国では、GoogleやBingの偽広告を通じて銀行のログイン情報を盗んでいた犯罪グループの管理ドメイン**「web3adspanels.org」**をFBIが押収した。このスキームによる実際の被害額は約1,460万ドルに達しており、数千人の認証情報がサーバーに保管されていたことが判明している,。

## 国内のインシデントと被害状況
国内では、**株式会社テーオーシー**や**埼玉県商工会連合会**がネットワーク機器を経由した不正アクセスを受け、ファイルの一部が暗号化されるなどの被害を公表した,,。また、**日産自動車**は、システム開発委託先のRed Hatが受けた不正アクセスの影響で、福岡の販売会社を利用した顧客約21,000人の氏名、住所、メールアドレスなどが漏洩したことを確認した,。このほか、11月の集計では、アクサ損害保険におけるペット保険システムの不正アクセス（約55万件）が最大の被害規模となっている。

# 2025年12月24日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. Array Networks Array AGシリーズの脆弱性悪用に関する注意喚起
Array NetworksのAGシリーズに搭載されている「DesktopDirect」機能に、コマンドインジェクションの脆弱性が存在することが公表された。JPCERT/CCによれば、2025年8月以降にこの脆弱性を悪用した攻撃が国内で発生しており、Webshellの設置や新規ユーザーの作成、さらには同製品を経由した組織内部への侵入が確認されている,。修正済みバージョン（9.4.5.9）は既に公開されているが、適用時の再起動によりログが消失する恐れがあるため、事前に保全を行うことが推奨されている。
[出典: JPCERT/CC - Array Networks Array AGシリーズにおけるコマンドインジェクションの脆弱性に関する注意喚起](https://www.jpcert.or.jp/at/2025/at250024.html)

### 2. フランス郵便・銀行サービス「La Poste」がサイバー攻撃で停止
フランスの国営郵便サービス「La Poste」およびその銀行部門「La Banque Postale」において、大規模なネットワーク障害が発生し、デジタルサービスやオンラインバンキングが一時的に停止した。数百万人の顧客に影響が及んだが、窓口業務やATMでの現金引き出し、店舗でのカード決済は継続可能であった,。現地メディアの報道によれば、原因は分散型サービス拒否（DDoS）攻撃によるものとされている。
[出典: BleepingComputer - Cyberattack knocks offline France's postal, banking services](https://www.bleepingcomputer.com/news/security/cyberattack-knocks-offline-frances-postal-banking-services/)

### 3. 日産自動車、Red Hatのデータ侵害による顧客情報漏洩を確認
日産自動車は、顧客管理システムの開発を委託していたRed Hat社のサーバーが第三者からの不正アクセスを受け、データが漏洩したとの報告を受けた,。調査の結果、日産福岡販売の顧客約21,000人分の氏名、住所、電話番号、メールアドレスが含まれていたことが判明した。漏洩の原因はRed Hatが利用していたGitLabリポジトリの窃取に関連しているとされ、他国の日産拠点でも過去にAkiraランサムウェアなどの被害が発生していた背景がある,。
[出典: BleepingComputer - Nissan says thousands of customers exposed in Red Hat breach](https://www.bleepingcomputer.com/news/security/nissan-says-thousands-of-customers-exposed-in-red-hat-breach/)

## 深掘り

### ネットワーク境界機器を起点とした侵入拡大
最近のインシデントでは、VPNやリモートアクセスを提供するネットワーク機器そのものが、組織内部へ侵入するための「最初の足場」として執拗に狙われている。

#### Array AGシリーズの脆弱性悪用の詳細
Array AGシリーズのケースでは、攻撃者はDesktopDirect機能の不備を突き、OSコマンドを直接実行するコマンドインジェクション攻撃を仕掛けている。特筆すべきは、国内の複数の事例において、攻撃者が "/webapp/" ディレクトリの下にPHPファイル形式の**Webshell**を設置しようとしていた点である。これにより、攻撃者は認証を回避して継続的にサーバーを遠隔操作し、内部ネットワークの偵察やデータ窃取が可能になる。

#### 物理的な隙とデジタル被害の交差
11月の国内事例の総括によれば、技術的な脆弱性だけでなく、人的・物理的な要因による漏洩も深刻である。例えば、中学校教員が飲酒後に路上で寝ている間に、個人情報を含むHDDやUSBメモリ、端末を紛失する事案が発生し、SNS等で大きな注目を集めた。また、地方病院の患者情報がUSBメモリから流出した際、その発覚のきっかけが「差出人不明の匿名郵便」であったというミステリーのような事例も報告されている。

### 関連知識

#### 用語解説
* **WebRAT:** スチームやディスコードの資格情報、仮想通貨ウォレットを盗む機能を持つバックドアマルウェア。ウェブカメラによるスパイ行為も可能。
* **PoC（Proof of Concept）:** 脆弱性の存在を証明するための概念実証コード。GitHub上でこのPoCを偽装してマルウェアを配布する手法が急増している。
* **Operation Sentinel:** インターポールがコーディネートした、アフリカにおけるサイバー犯罪掃討作戦。BEC（ビジネスメール詐欺）やランサムウェアの被害回復に成功した,。

#### 今後の展望
攻撃者はAI（人工知能）を駆使して、より自然で信頼性の高い攻撃用コンテンツを作成し始めている。GitHubの偽リポジトリに見られるように、技術者であっても一見して偽物と判断しにくい高度な偽装が今後も増加するだろう。組織は「信頼できるサイトからのダウンロード」であっても、実行前に隔離された環境（サンドボックス）で検証する習慣を徹底し、境界防御だけに頼らないゼロトラストなセキュリティモデルへの移行を加速させる必要がある。

## 結論
2025年12月のニュースは、国際協力による犯罪インフラの解体という明るい兆しがある一方で、国内では依然として境界防御機器の脆弱性や委託先の管理不備を突く攻撃が猛威を振るっていることを示している,,。パッチ適用やMFA（多要素認証）の導入といった基本対策に加え、外部ベンダーを含めたサプライチェーン全体のセキュリティ状況を可視化・監査することが、現代の企業防衛において不可欠な要素となっている。
