## 📖 解説パート

### リスクベース思考とは

リスクベース思考とは、組織が目標を達成する上で影響を与える可能性のあるリスクと機会を考慮し、それらに対応するための体系的なアプローチです。単にネガティブな側面を避けるだけでなく、ポジティブな機会を最大限に活用することも目的としています。ISO 9001:2015では、このリスクベース思考が品質マネジメントシステムの基盤として組み込まれています。

リスクは、不確実性によって生じる影響であり、目標達成を阻害する可能性を持つものです。機会は、好ましい状況や可能性であり、組織のパフォーマンス向上に貢献するものです。リスクベース思考は、これらのリスクと機会を特定し、評価し、対応策を講じるプロセスを指します。

### ISO 9001とリスクベース思考

ISO 9001:2015では、リスクベース思考が明示的に要求事項として規定されていません。しかし、規格全体を通して、リスクと機会を考慮した活動が求められています。例えば、以下の点が挙げられます。

* **4.1 組織及びその状況の理解:** 組織を取り巻く内部及び外部の課題を決定する際に、リスクと機会を考慮する必要があります。
* **6.1 リスク及び機会への取組み:** 品質マネジメントシステムが意図した結果を達成できることを確実にするために、リスクと機会に取り組むための計画を立てる必要があります。
* **8.1 運用の計画及び管理:** 製品及びサービスの提供プロセスの計画、実施、管理において、リスクを考慮する必要があります。

リスクベース思考をISO 9001に適用することで、品質マネジメントシステムの効果を高め、顧客満足度を向上させ、組織の持続的な成長を促進することができます。

```
# リスクアセスメントの例 (簡略化)
risk_name = "サプライヤの品質問題"
likelihood = "中"  # 高、中、低
impact = "大"      # 大、中、小

if likelihood == "高" and impact == "大":
    risk_level = "高"
elif likelihood == "中" and impact == "大":
    risk_level = "中"
else:
    risk_level = "低"

print(f"リスク名: {risk_name}, リスクレベル: {risk_level}")
```

#### 重要ポイント

* リスクベース思考は、リスクと機会の両方を考慮する。
* ISO 9001:2015は、リスクベース思考を品質マネジメントシステムの基盤としている。
* リスクアセスメントを通じて、リスクレベルを評価し、適切な対応策を講じることが重要。

## 🏢 ケーススタディ

### ケース：自動車部品メーカーにおけるサプライチェーンリスク

自動車部品メーカーA社は、海外のサプライヤから部品を調達していました。ある時、サプライヤの品質管理体制に問題が発生し、不良部品が多数納入されるという事態が発生しました。これにより、A社の生産ラインが停止し、顧客への納期遅延が発生しました。

#### 問題点

* サプライヤの選定基準が不十分であった。
* サプライヤの品質管理体制に対する監査が不十分であった。
* サプライチェーン全体のリスク評価が不足していた。

#### 対応策

* サプライヤ選定時に、品質管理体制に関する詳細な評価を行う。
* 定期的なサプライヤ監査を実施し、品質管理体制の改善を支援する。
* サプライチェーン全体のリスクを評価し、代替サプライヤの確保などの対策を講じる。
* 不良部品の早期発見と対応のための検査体制を強化する。

#### ケースから学ぶ教訓

* サプライチェーン全体のリスクを常に意識し、評価・管理することが重要。
* サプライヤとの緊密な連携と継続的な改善活動が不可欠。
* リスク発生時の迅速な対応体制を整備しておくことが重要。

### 問題1 正解：1

### 問題2 正解：3

解説：ISO 9001:2015では、リスクベース思考が明示的に要求事項として規定されていませんが、規格全体を通して、リスクと機会を考慮した活動が求められています。例えば、組織の状況の理解、リスク及び機会への取組み、運用の計画及び管理などが挙げられます。

### 問題5 正解：2

解説：リスクベース思考を品質マネジメントシステムに適用することで、品質マネジメントシステムの効果を高め、顧客満足度を向上させ、組織の持続的な成長を促進することができます。リスクと機会を考慮した活動は、組織のパフォーマンス向上に貢献します。
