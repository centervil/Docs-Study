## 📖 解説パート

### データの種類と特徴

データは、大きく分けて量的データと質的データに分類されます。

*   **量的データ:** 数値で表現されるデータ。例えば、製品の長さ、重さ、温度など。量的データはさらに、連続データ（小数点以下を含む数値）と離散データ（整数値）に分けられます。
*   **質的データ:** カテゴリや属性で表現されるデータ。例えば、製品の色、欠陥の種類、顧客の満足度（非常に満足、満足、普通、不満、非常に不満）など。質的データは、名義尺度（単なる区別）と順序尺度（大小関係がある）に分けられます。

### 代表値：データの中心を捉える

代表値は、データセット全体を代表する値であり、データの中心的な傾向を示す指標です。代表的なものとして、平均値、中央値、最頻値があります。

*   **平均値:** 全てのデータの値を合計し、データ数で割った値。外れ値の影響を受けやすいのが欠点です。
*   **中央値:** データを小さい順に並べたとき、ちょうど真ん中に位置する値。外れ値の影響を受けにくいという利点があります。データ数が偶数の場合は、中央に位置する2つの値の平均を取ります。
*   **最頻値:** データの中で最も頻繁に出現する値。データがカテゴリカルな場合（例えば、製品の色）に特に有用です。

### ばらつき：データの散らばり具合を測る

ばらつきは、データがどの程度散らばっているかを示す指標です。代表的なものとして、分散と標準偏差があります。

*   **分散:** 各データと平均値の差の二乗を合計し、データ数で割った値。データのばらつきの大きさを数値で表します。
*   **標準偏差:** 分散の平方根。分散よりも直感的にデータのばらつきを把握しやすいです。

```python
import numpy as np

data = [10, 12, 15, 13, 11, 14, 16, 12, 13, 15]

# 平均値
mean = np.mean(data)
print(f"平均値: {mean}")

# 中央値
median = np.median(data)
print(f"中央値: {median}")

# 標準偏差
std = np.std(data)
print(f"標準偏差: {std}")

# 分散
variance = np.var(data)
print(f"分散: {variance}")
```

#### 重要ポイント

*   データの種類に応じて適切な代表値とばらつきの指標を選択することが重要です。
*   外れ値は、代表値やばらつきに大きな影響を与える可能性があるため、注意が必要です。
*   標準偏差は、データの単位と同じ単位で表現されるため、解釈しやすいです。

## 🏢 ケーススタディ

### ケース：製造ラインにおける製品の長さのばらつき

ある製造ラインで製造された製品の長さが、仕様範囲内に収まっているかを確認したいと考えています。しかし、測定された製品の長さにはばらつきがあり、規格外の製品が発生する可能性が懸念されています。

#### 問題点

*   製品の長さの平均値が仕様範囲内であっても、ばらつきが大きいと規格外の製品が発生する可能性がある。
*   ばらつきの原因が特定できていないため、対策を講じることが難しい。
*   どの程度ばらつきを抑えれば良いのか、具体的な目標値が設定できていない。

#### 対応策

*   製品の長さを測定し、平均値、標準偏差、工程能力指数（Cp, Cpk）を算出する。
*   管理図を作成し、工程の安定性を確認する。異常な変動が見られる場合は、原因を特定し対策を講じる。
*   実験計画法などを活用し、製品の長さに影響を与える要因を特定する。
*   製造プロセスの改善を行い、ばらつきを低減する。
*   定期的に製品の長さを測定し、工程能力を監視する。

#### ケースから学ぶ教訓

*   平均値だけでなく、ばらつきも考慮することで、より正確な品質管理が可能になる。
*   データに基づいた分析を行うことで、問題の原因を特定し、効果的な対策を講じることができる。
*   継続的な改善活動を行うことで、品質の安定化を図ることができる。

## ✅ テスト回答・解説

### 問題1 正解：3

解説：製品の色は、数値で表現されず、カテゴリや属性で表現されるため、質的データに該当します。製品の長さ、重さ、温度は、数値で表現される量的データです。

### 問題2 正解：3

解説：データセット[2, 4, 6, 8, 10]は、すでに小さい順に並んでいるため、中央に位置する値は6です。

### 問題3 正解：2

解説：標準偏差はデータの散らばり具合を示す指標であり、標準偏差が大きいほど、データが平均値から大きく離れているものが多いことを意味します。

### 問題4 正解：2

解説：正規分布において、平均値±標準偏差の範囲に約68%のデータが含まれます。したがって、95mm～105mmの範囲に約68%の製品の長さが収まります。

### 問題5 正解：2

解説：中央値は、データを小さい順に並べたときの中央に位置する値であるため、外れ値の影響を受けにくいです。平均値は、全てのデータの値を合計して計算するため、外れ値の影響を受けやすいです。最頻値は、データの中で最も頻繁に出現する値であるため、外れ値が頻繁に出現する場合は影響を受ける可能性があります。分散は、各データと平均値の差の二乗を合計して計算するため、外れ値の影響を受けやすいです。
